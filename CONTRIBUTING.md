# Contributing To IIL 

First off, thanks for contributing and supporting this project! 

All types of contributions are allowed and greatly valued. 
For different ways to help and details about how this project handels contributions, see the [Table of Contents](#table-of-contents). 
It will make it easier for mainteners, and it will smooth out the experience for everybody. 

> If you like the project, but don't have the time or patience (lol), that's okay. There are other ways to support the project, which are greatly appreciated:
> - Star the project
> - Tweet about the project
> - Refer this project in your project's readme
> - Mention this project to friends.

## I Have a Question
If you have a question, I assume you've already read the available documentation. I also assume you've already asked google (lol). 
Anyway, if you have a question just: 
- Open an [Issue](/issues/new)
- Provide context.
- Provide the project version and platform you're running.

I'll try to respond to questions as soon as possible. 
> [!NOTE]
> You can also send me an email: otso.reijonen@icloud.com
